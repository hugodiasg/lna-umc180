# lna-umc180
Low Noise Amplifier (LNA) with degeneration inductor to 2.38 GHz - 2.43 GHz band in UMC 180nm.

## Specs
Parameter   | Value
--------- | ------
Frequency | 2.38 - 2.42 GHz
Gain (S21) | >18.2 dB
Return Loss (S11) | <-22.9 dB
Noise Figure | <0.5 dB
Output Impedance | 5 kÎ©
